  Just copy/paste these commands in order

STEP 1: CREATE GITHUB REPOSITORY
─────────────────────────────────
1. Go to github.com and log in
2. Click the + icon → New repository
3. Name it: therapist-admin-mvp
4. Click "Create repository"
5. COPY the HTTPS URL (looks like: https://github.com/YOUR_USERNAME/therapist-admin-mvp.git)


STEP 2: OPEN TERMINAL/COMMAND PROMPT
────────────────────────────────────
On Windows: Press Win+R, type "cmd", hit Enter
On Mac: Press Cmd+Space, type "terminal", hit Enter


STEP 3: NAVIGATE TO YOUR PROJECT FOLDER
───────────────────────────────────────
Copy/paste this (Windows):
  cd C:\Users\luker\Documents\Claude\Projects\"Digital Products"

Or Mac/Linux:
  cd ~/Documents/Claude/Projects/"Digital Products"


STEP 4: RUN THESE COMMANDS (one at a time)
──────────────────────────────────────────

git init

git config user.name "Luke Ryan"

git config user.email "lukeryanrose762@gmail.com"

git add .

git commit -m "Initial therapist admin MVP"

git remote add origin https://github.com/LukeRyan31/therapist-admin-mvp.git

git branch -M main

git push -u origin main


STEP 5: VERIFY
──────────────
1. Go to https://github.com/YOUR_USERNAME/therapist-admin-mvp
2. You should see all your files there
3. You're DONE! ✅

  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PROBLEM: "git: command not found"
SOLUTION: Download git from git-scm.com

PROBLEM: Permission denied when pushing
SOLUTION: Go to github.com/settings/tokens → Generate token → Use token as password

PROBLEM: Can't find your folder
SOLUTION: Right-click folder → "Open in Terminal" or "Open PowerShell here"
